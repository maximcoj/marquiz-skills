# Landing Writer

Генератор структуры и текстов лендинга на основе ценностного предложения.

## Зависимости

Скилл использует выход от [value-prop-writer](../value-prop-writer/) — документ с 8 инвариантами ценностного предложения по модели ДКЦП.

## Что делает

1. Принимает готовое ценностное предложение
2. Трансформирует инварианты в структуру лендинга из 4 экранов
3. Генерирует тексты в профессиональном тоне
4. Даёт рекомендации по визуалу

## Структура лендинга

| Экран | Цель | Источники из value-prop |
|-------|------|------------------------|
| **Hero** | Захват внимания, главная ценность | Аргументы "Результат"/"Решение", Артефакт, Атрибуты компании |
| **Проблема и решение** | Резонанс с ситуацией клиента | Мотивационный конфликт, Ключевые атрибуты |
| **Возможности** | Конкретные функции продукта | Уникальные и ключевые атрибуты артефакта |
| **Доверие и действие** | Снятие возражений, CTA | Атрибуты компании, Условия обмена, Вклад клиента |

## Особенности тона

- Обращение на «вы»
- Профессиональный язык без манипуляций
- Конкретика вместо эмоций
- Метрики вместо абстракций

## Использование

```
@landing-writer Создай лендинг на основе @value-prop-writer/output-marketing-directors.md
```
